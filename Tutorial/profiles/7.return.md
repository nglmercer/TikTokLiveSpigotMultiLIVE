
# 8. What is Return

Return is magic keyword that ends execution of code or set output value (for functions)

it can be use in

-  Events
``` yml
onGift:

  if event.gift.name is "rose"
   return
   
  /say hello world  
```   

- Functions
``` yml

getName: (value) then
     if value is 1
     then return "mark"
     else return "adam"

#use return to stop execution of function
calucalte: (value) then
     value = 0
     value = 1
     value = 2
     value = 4
     return

onGift:
  /say hello world ${getName(1)}  
  calucalte()
```
